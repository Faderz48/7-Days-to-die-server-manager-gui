NETWORK / PORT FORWARDING

Click auto-forward to ask your router (via UPnP) to open ServerPort, ServerPort+1, and ServerPort+2 on both TCP & UDP. Required for friends outside your home network to connect.

WINDOWS FIREWALL

Even with the router forwarding ports, Windows Firewall on this PC will silently drop incoming connections unless they're explicitly allowed. Click allow in firewall to add rules for ServerPort, +1, and +2 (TCP & UDP). Requires running this manager as administrator.

Can't port forward? Use a virtual LAN instead

If port forwarding isn't an option (CGNAT ISP, locked-down router, office network), you can use a virtual LAN tool. Both you and your friends install the same one, join the same network, and then the server appears to everyone as if it was on a real LAN — no router config, no public IP needed.

  • Radmin VPN

    Free, no player cap, simple. Best first choice for most people.

    download

  • Hamachi (LogMeIn)

    Old standby. Free up to 5 players per network, easy to walk friends through.

    download

  • Tailscale

    Modern, fast, free for personal use. Requires each player to log in.

    download

  • ZeroTier

    Open-source, free up to 10 nodes. More configurable but slightly fiddlier.

WORLD

Select the map this server runs. Pre-made maps and any random worlds you've already generated will appear here.

SEED VAULT

Generate readable, repeatable seed names. Click one to set it as the active WorldGenSeed. Identical seeds produce identical worlds — the actual world generates on first server start.

    WORLD TRANSFER

    Download an existing save folder as a single .zip, or upload a .zip to drop a pre-built world straight onto the server. Uploaded worlds extract into the saves directory and become available to GameName right after.

    UPLOAD A WORLD

    Pick a .zip file. If the zip's top-level folder is named Navezgane, the save lands at <saves_dir>/Navezgane. Use the override field to rename it.

    MODS

    Drop a modlet zip in here and it extracts straight into the server's Mods/ folder. Most mods on Nexus, the 7DTD forums, or the Mod Launcher come zipped already — upload as-is. The extractor unwraps double-wrapped zips automatically (e.g. MyMod-v1.2/MyMod/).

    SAVED CONFIG PRESETS

    Save the current serverconfig.xml under a name so you can swap between, say, a PvE and PvP setup with one click. Each preset also has export and import — handy for sharing a tuned config with friends or backing one up off-machine.
